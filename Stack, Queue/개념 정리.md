### 스택 자료구조
+ 먼저 들어 온 데이터가 나중에 나가는 형식
+ 입구와 출구가 동일한 형태

```python
stack = []

stack.append(5)
stack.append(2)
stack.append(3)
stack.append(7)
stack.pop()
stack.append(1)
stack.append(4)
stack.pop()

print(stack[::-1]) # 최상단 원소부터 출력   [1, 3, 2, 5]
print(stack) # 최하단 원소부터 출력  [5, 2, 3, 1]
```
```c++
stack<int> s;

int main(void) {
    s.push(5);
    s.push(2);
    s.push(3);
    s.push(7);
    s.pop();
    s.push(1);
    s.push(4);
    s.pop();
    
    // 스택의 최상단 원소부터 출력
    while (!s.empty()) {
        cout << s.top() << ' ';
        s.pop();
    }
}

// 결과: 1, 3, 2, 5
```
### 큐 자료구조
+ 먼저 들어 온 데이터가 먼저 나가는 형식
+ 입구와 출구가 모두 뚫려 있는 터널과 같은 형태

```python
from colletions import deque

queue = deque()  # deque = 데이터를 양방향에서 추가하고 제거할 수 있는 자료 구조

queue.append(5)
queue.append(2)
queue.append(3)
queue.append(7)
queue.popleft()
queue.append(1)
queue.append(4)
queue.popleft()

print(queue)    # 먼저 들어온 순서대로  deque([3, 7, 1, 4])
queue.reverse()
print(queue)    # 나중에 들어온 원소부터 deque([4, 1, 7, 3])
```
```python
queue = deque([1, 2, 3, 4, 5])

# 요소 추가
queue.append(x)      # 오른쪽에 요소 추가
queue.appendleft(x)  # 왼쪽에 요소 추가


# 요소 제거
queue.pop()      # 오른쪽 끝 요소 제거
queue.popleft()  # 왼쪽 끝 요소 제거

# 요소를 n칸씩 오른쪽으로 이동
queue.rotate(1)   # [5, 1, 2, 3, 4]
queue.rotate(-1)  # [2, 3, 4, 5, 1]
```
